Web scraping with Python

Definition

Web scraping with Python is the use of an algorithm written with Python to extract and process large amounts of data from the web.

The project

I wrote a web scraping code in Python to obtain an automatically generated excel with KPI wanted about world countries from the World Factbook. As you can see in the figure below, the output is an excel sheet with all the KPI selected by the user (in blue) about each country in the world (in yellow). In this form the data (in green) can be easily exploited by people.

Why this project?

I thought about this project when I was doing an internship as a strategic analyst in a company. I observed that some people needed to create and update data relating to a number of countries in order to enable the relevant people to make strategic decisions, particularly in terms of international development.

The process was time consuming and very repetitive. For each country people had to find, copy and past all information needed in an excel document. It was not efficient to waste time on such repetitive processes for employees who could be working on tasks that require their special skills.

This finding led me to want to learn how to automate such tasks.
